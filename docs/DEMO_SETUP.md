# TazaBAK: чек-лист демонстрации MVP

Этот документ помогает подготовить TazaBAK к показу на защите.

## 1. Запуск платформы

1. Запустите FastAPI на ноутбуке: `uvicorn app.main:app --reload --host 0.0.0.0 --port 8000`.
2. Откройте второй терминал, перейдите в `frontend` и выполните `npm run dev -- --host 0.0.0.0`.
3. В браузере откройте адрес Vite. Для телефона используйте сетевой адрес ноутбука, а не `localhost`.
4. Откройте диспетчерскую и убедитесь, что API отмечено как доступное.

## 2. Муниципальный IoT-контур

- ESP32 DevKit отправляет расстояние HC-SR04 и температуру DS18B20 в FastAPI по Wi-Fi.
- Заполнение рассчитывается по расстоянию: 25 см — пустой бак, 7 см — полный бак.
- Температура внутри выше 50 C создаёт критический алерт FIRE_RISK.
- ESP32-CAM передаёт MJPEG-поток; FastAPI сохраняет кадр с рамками YOLOv8 при обнаружении незаконного навала.
- Команда CLOSE_LID доставляется ESP32 через WebSocket и закрывает SG90.

## 3. Принцип работы

1. Житель фотографирует сухой хлеб и получает решение, QR-код и Eco-баллы.
2. Диспетчер видит состояние контейнера и изображение с ESP32-CAM.
3. Камера фиксирует посторонний объект у бака, появляется ILLEGAL_DUMP.
4. Волонтёр берёт задачу по уборке и получает баллы после подтверждения.
5. При переполнении диспетчер отправляет команду закрытия заслонки.
